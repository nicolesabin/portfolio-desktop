Child Care Website Redesign

UX Design

Skills Used:

  • Web Design
  • UI/UX Design
  • Technical Writing
  • HTML/CSS

Project Overview

The Weber State Child Care website was redesigned to improve the user experience and make it more intuitive for parents to find and enroll their children in the WSU child care program. The redesign included updated photos, a new color scheme, and a new layout

Before the redesign, the website was difficult to navigate and the information was not organized in a user-friendly way. Even though the child care program offered affordable care for the children of our students while they attended classes, the website was a huge pain point for them and many didn't complete the enrollment process. As a result, the child care program had many vacancies each semester.The director of the child care program was frustrated that their program was very underutilized and wanted to improve the experience for Weber State students.

Project Details

This project involved comprehensive web design processes:

  • User research to understand target users by looking through branding materials and talking to the director of the child care program about the steps a parent would take to enroll their child in the program
  • Information architecture and updating the copy of the website to be more simple and concise
  • Prototyping the new website in Weber State's website builder using HTML and CSS
  • Collaborating with marketing professionals and the child care program director to ensure the new website was a success
  • Taking the website live and creating redirects from the old website to the new website
  • Continuous iterative design process based on user feedback and changes to the program.

Key Achievements

For the first time in years, the child care program had enough parents interested in the program that they were able to fill all of its vacancies each semester AND a waitlist was created for the overflow of students. The new website was a success and the child care program has received positive feedback from the parents that utilize their services.
