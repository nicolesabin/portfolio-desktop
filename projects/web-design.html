Project Overview

This web design project demonstrates my expertise in web design, from research and user persona development to wireframing and prototyping using industry-standard tools.

This project involved comprehensive web design processes:

  • User-centered design principles application
  • Iterative design process based on user feedback

Key Achievements

This project showcases proficiency in the complete web design workflow, from initial research through final design deliverables. The use of tools like Lucidchart for diagramming and Adobe XD for prototyping demonstrates technical competency, while the focus on user personas shows a deep understanding of user-centered design principles.
