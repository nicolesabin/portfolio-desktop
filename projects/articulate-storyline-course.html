Articulate Storyline Course

eLearning

Skills Used:

  • Articulate Storyline
  • Graphics Creation
  • Amazon Web Services

Project Overview

This interactive e-learning course was created for onboarding new lifeguards. The course uses gamification principles to engage learners, allowing them to follow along as they learn how to be a lifeguard while collecting badges in their fanny pack.

Open course in new window

Project Details

This project showcases my expertise in Articulate Storyline and gamification principles. The course features:

  • Interactive learning modules with engaging content
  • Badge collection system to track learner progress
  • Custom graphics and visual design
  • Hosting and deployment using Amazon Web Services
  • User-friendly navigation and intuitive interface

Key Features

The gamification element of collecting badges in a fanny pack creates a fun and memorable learning experience that motivates learners to complete the course. This approach demonstrates how interactive elements can enhance learning outcomes and engagement.
